04

How it's made

The honest pipeline — shared openly, because directing AI well is the actual skill.

01

Direct, don't micromanage

I give the agent intent and constraints — "anime comic, my website, make it live" — and approve at checkpoints: storyline, art style, publish. High-level direction, verifiable outcomes.

02

Lock the creative system

Consistency comes from engineering: frozen character sheets and one style string — 1990s anime cel · neon-noir Tokyo · cyan/magenta — injected verbatim into every image-generation call.

03

Build resilient pipelines

Async background jobs for 12 parallel renders, polling within sandbox timeouts, automatic retries, web-optimized output (35MB → 5MB). When the environment fights back, the pipeline adapts.

04

Ship autonomously, verify ruthlessly

The agent drives a real browser to commit to GitHub, then verifies the live URL, the rendered DOM, and the pixels before calling it done. No deploy is "done" until production says so.
